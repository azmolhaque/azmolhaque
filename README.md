### 🔎 Featured security research
**[Exposed IAM Front-End on a Google-Owned Domain — Google VRP](https://github.com/azmolhaque/security-writeups/blob/main/2026-05-exposed-iam-frontend-google-vrp.md)**
A default-credential + any-password login on a forgotten admin panel, backed by an
unauthenticated API — reachable *through* Google's edge but never *operated* by Google.
Reported via Google Bug Hunters and remediated in 9 days. I rated it accurately; it
resolved to low real-world impact and was **correctly credit-only**. The write-up covers
the repeatable discovery method, the "edge-fronted ≠ operated" root-cause distinction, and
a defender's detection & remediation plan — in 9 languages.
